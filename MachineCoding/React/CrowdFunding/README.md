# CrowdFunding

A minimal crowdfunding app built with React + TypeScript + Vite.

![Preview](@preview/01.png)

## Features

- Multiple campaigns displayed in a responsive grid
- Progress bar showing how much has been raised toward the goal
- Contribute form — enter your name and amount to join a campaign pool
- Withdraw — any contributor can remove their contribution
- Fully Funded badge when a campaign reaches its goal

## Stack

- React 19
- TypeScript
- Vite

## Getting Started

```bash
npm install
npm run dev
```
