## Job Summary: 
Wissen Technology is hiring a Senior React Native Developer with strong expertise in mobile application development and native integrations. 

* **Experience:** 5-8 Years 
* **Location:** Bangalore (Local only) 
* **Mode of Work:** Hybrid (Thrice a week from office mandatory) 

## Key Responsibilities: 

* Develop and maintain mobile applications using React Native, JavaScript (ES6+), and TypeScript. 
* Implement state management using libraries like Redux and Zustand. 
* Configure and manage navigation flows using react-navigation and deep linking. 
* Work with Expo in both managed and bare workflows. 
* Integrate native modules and UI components using Swift/Objective-C (iOS) and Java/Kotlin (Android). 
* Bridge custom native modules and ensure seamless native integration. 
* Debug applications using tools like Flipper and Chrome DevTools. 
* Write unit and integration tests using Jest and React Native Testing Library. 
* Integrate REST APIs, GraphQL, and real-time data using WebSockets. 
* Manage OTA updates with EAS Update and configure EAS Build. 
* Handle app publishing and release lifecycle for both Play Store and App Store. 
* Implement secure storage, token handling, and app-level security best practices. 

## Required Skills and Qualification: 

* Proficient in React Native, JavaScript (ES6+), and TypeScript. 
* Strong experience with Redux, Zustand, or similar state management libraries. 
* Hands-on experience with react-navigation and deep linking. 
* Solid understanding of Expo workflows. 
* Knowledge of native development for iOS and Android. 
* Experience with custom native module bridging. 
* Familiarity with debugging tools like Flipper and Chrome DevTools. 
* Experience with Jest and React Native Testing Library. 
* Strong understanding of REST APIs, GraphQL, and WebSockets. 
* Experience with EAS Update and EAS Build. 
* Proven experience in app publishing and release management. 
* Knowledge of secure storage and app-level security practices.  

## Good to Have Skills: 

* Experience working in Agile development environments. 
* Exposure to CI/CD pipelines for mobile apps. 
* Familiarity with performance optimization techniques for mobile apps. 
* Strong communication and collaboration skills.
