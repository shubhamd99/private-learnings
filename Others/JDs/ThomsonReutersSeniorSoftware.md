# Thomson Reuters - Senior Software Engineer (Mobile)

**Job Requisition ID:** JREQ196790

The Client Application Technology team is dedicated to creating compelling experiences that deliver Reuters' outstanding journalism to the world’s most informed professionals. The team aims to increase discovery, optimise conversion, and drive engagement across Thomson Reuters products and services. The technology focuses on web development and substantial backend service integration.

You will work with a distributed team of software developers, quality engineers, and product managers focused on developing and integrating platforms and services that power Reuters experiences, including content creation, curation, publishing, distribution, and e-commerce. 

## About the Role:

As a Mobile Senior Software Engineer at Reuters, you will: 

* Lead the technical design and development for Reuters' React Native cross-platform mobile application.
* Implement new features for the React Native mobile application by integrating headless CMS and other backend services.
* Develop React Native components and screens following best practices and design patterns.
* Develop and execute unit and integration testing using React Native Testing Library, ensuring software meets quality assurance requirements.
* Coordinate with internal teams and stakeholders and contribute to requirements analysis, product design, and architecture definition.
* Develop a disciplined approach, following established software development processes, adhering to organizational standards, software best practice guidelines, and corporate policies.
* Communicate clearly and effectively via GitHub, Slack, MS Teams, ADO etc., to keep all stakeholders in the loop on relevant issues.
* Architect a React Native cross-platform application to maximize code reuse across iOS and Android platforms.
* Participate in production troubleshooting and remediation for React Native applications.
* Participate in Agile ceremonies and software development lifecycles, including code review and deployment.
* Conduct regular code reviews to ensure a quality product.

## About You:

* 6-8 years of experience in mobile application development with at least 5 years of hands-on React Native development building production applications.
* Demonstrable portfolio of React Native applications released on Apple App Store and Google Play Store.
* Proven expertise in React Native design patterns, including hooks, context API, navigation, and state management.
* Strong proficiency in JavaScript and TypeScript within the React Native ecosystem.
* Deep understanding of React Native bridge and native modules, with working knowledge of Kotlin and Swift for implementing platform specific features and troubleshooting native issues.
* Strong Computer Science fundamentals, including data structures and algorithms.
* Strong understanding of software engineering best practices, including coding patterns, code reviews, source control management, build processes, testing, and monitoring.
* Experienced in comprehensive CI/CD-based React Native workflows, including automated builds, unit testing, beta distribution, and seamless deployment to app stores.
* Proficiency in working with APIs, including hands-on experience designing, consuming, and integrating RESTful and GraphQL APIs into React Native applications.
* Experience with React Native third-party libraries and managing dependencies.
* Familiarity with React Native development tools and technologies such as React Native CLI, Metro bundler, Flipper, VSCode, Xcode, Android Studio, Gradle, CocoaPods, and npm/yarn.
* Experience with React Native debugging tools and performance optimization techniques.
* Working knowledge of software development methodologies like Agile, Scrum, and Kanban.
* Familiarity with Content Management Systems (CMS) and their integration with React Native applications.
* Solid understanding of Design Systems, UX (User Experience), and UI (User Interface) principles in React Native context.
* Knowledge of in app advertising and digital subscriptions implementation in React Native.
* Expertise in React Native application performance benchmarking, optimization, and monitoring using tools like Flipper and crash reporting platforms.
